# Phase 1: Foundation & Chat UI

## Goal
Establish the core project architecture and build a functional, responsive Chat UI for students.

## Functional Requirements
- **Responsive Chat Interface:** A modern, "WhatsApp-style" or "ChatGPT-style" chat window.
- **Admin Dashboard UI:** A secure-ready layout for uploading documents and managing notices.
- **Project Scaffold:** Next.js frontend and FastAPI backend boilerplate.
- **Basic Routing:** Navigation between the Chat view and the Admin view.
- **Mock Interaction:** Chat UI handles local message state before backend integration.

## Technical Scope
- **Frontend:** Next.js (App Router), Tailwind CSS, Lucide React (icons).
- **Backend:** FastAPI, Python 3.10+.
- **State Management:** React `useState` for immediate UI feedback.
